Basic spectral pathtracer in zig 0.12.0
Build and run with:
```
git clone https://github.com/StrandedSoftwareDeveloper/spectralTracer.git
cd spectralTracer
zig build run
```
It then produces `out.png`

Planned features:
* CPU only
* f64
* Rayleigh scattering
* Atmospheric refraction?
* More stuff probably
